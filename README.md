---

## Features

- 🧮 Selling price + GST breakup calculator
- 3 profit modes — ₹ Amount, % Margin, % Markup
- CGST + SGST alag-alag dikhata hai
- ITC (Input Tax Credit) automatically calculate hota hai
- Margin health meter — green/yellow/red
- Bulk calculator — quantity × price = total profit
- 📋 History — calculations save hoti hain phone pe
- 📴 100% offline kaam karta hai — internet nahi chahiye
- 📱 Install on home screen — bilkul real app jaisa

---

## Supported GST Rates

| Rate | Products |
|------|----------|
| 0%   | Exempt items |
| 5%   | Basic electrical items |
| 12%  | Some electrical goods |
| 18%  | MCB, Switches, Wires, Fans |
| 28%  | Premium items |
